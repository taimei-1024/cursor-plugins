# 安全自查

请使用 tech-design-review 技能执行安全自查。

**使用方式**：
- `/aitd-security-check <设计文档内容>`
- `/aitd-security-check`

**输出内容**：
- 各类别覆盖情况统计
- 需补充项的具体建议
- 高风险项的修复优先级

**检查类别**：
- 注入防护（SQL / XSS / 命令注入）
- 认证与授权（接口鉴权、数据权限、越权防护）
- 数据保护（加密、脱敏、分类）
- 日志与审计（操作日志、日志脱敏）
- 接口安全（参数校验、频率限制、幂等性）
- 业务安全（并发控制、状态机、数据一致性）
- 基础设施安全（配置安全、依赖安全）
