# 分析 AxShare 原型

请使用 axshare 技能分析指定的 AxShare 原型页面。

**使用方式**：
- `/aitd-analyze-prototype https://xxx.axshare.com`
- `/aitd-analyze-prototype`

**输出内容**：
- 原型页面截图
- 文本识别结果
- 交互说明
- 字段清单
- 结构化原型说明书
