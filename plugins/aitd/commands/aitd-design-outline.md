# 生成设计大纲

请使用 tech-design-review 技能生成技术设计文档大纲。

**使用方式**：
- `/aitd-design-outline` - 根据需求类型自动选择模板
- `/aitd-design-outline 总体设计` - 总体设计模板
- `/aitd-design-outline 后端详细设计` - 后端详细设计模板
- `/aitd-design-outline 前端详细设计` - 前端详细设计模板

**模板选择**：
- **总体设计**：新功能开发、系统架构设计、重大变更
- **后端详细设计**：单功能开发、接口开发、小范围迭代
- **前端详细设计**：前端页面开发、交互设计、组件开发

**输出内容**：
- 标准技术设计文档大纲
- 按模板组织的章节结构
- 必写/选写章节标注

如果需要，可以先询问用户需求类型和模板选择。
