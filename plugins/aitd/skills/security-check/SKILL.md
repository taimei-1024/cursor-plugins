# 安全漏洞自查

基于 [security-checklist.md](../tech-design-review/security-checklist.md) 对技术设计文档进行安全维度扫描。

## 执行方式

读取技术设计文档内容，逐项对照安全检查清单进行评估。

## 输出

安全自查报告，包含：
- 各类别覆盖情况统计
- 需补充项的具体建议
- 高风险项的修复优先级

## 覆盖类别

- **注入防护**（SQL / XSS / 命令注入）
- **认证与授权**（接口鉴权、数据权限、越权防护）
- **数据保护**（加密、脱敏、分类）
- **日志与审计**（操作日志、日志脱敏）
- **接口安全**（参数校验、频率限制、幂等性）
- **业务安全**（并发控制、状态机、数据一致性）
- **基础设施安全**（配置安全、依赖安全）
