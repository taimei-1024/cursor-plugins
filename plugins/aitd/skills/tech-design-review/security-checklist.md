# 安全自查清单

基于 OWASP Top 10 + 业务安全场景，用于技术设计安全自查。

---

## 使用方式

对照技术设计文档逐项检查，标注结果：
- ✅ **已覆盖** — 设计中已考虑
- ⚠️ **需补充** — 设计中未明确，建议补充
- ➖ **不适用** — 当前需求不涉及

---

## 1. 注入防护

| # | 检查项 | 说明 |
|---|--------|------|
| 1.1 | SQL 注入防护 | 是否使用参数化查询 / MyBatis #{} 占位符，避免 ${} 拼接 |
| 1.2 | XSS 防护 | 用户输入是否做 HTML 转义后再存储/展示 |
| 1.3 | 命令注入防护 | 是否避免将用户输入拼入系统命令 |
| 1.4 | LDAP / XML 注入 | 涉及 LDAP 或 XML 解析时是否做输入校验 |
| 1.5 | 日志注入防护 | 用户输入写入日志前是否过滤换行符和特殊字符 |

## 2. 认证与授权

| # | 检查项 | 说明 |
|---|--------|------|
| 2.1 | 接口鉴权 | 所有接口是否都经过 Token 校验（除白名单接口） |
| 2.2 | 功能权限 | 接口是否关联功能权限码，角色无权限时拒绝访问 |
| 2.3 | 数据权限 | 查询/操作数据时是否校验 tenant_id / project_id 隔离 |
| 2.4 | 越权防护 | 是否防止水平越权（A 用户操作 B 用户数据）和垂直越权 |
| 2.5 | Token 安全 | Token 是否设置合理过期时间，是否支持主动失效 |

## 3. 数据保护

| # | 检查项 | 说明 |
|---|--------|------|
| 3.1 | 敏感数据加密 | 密码、密钥等是否加密存储（不可逆加密/对称加密） |
| 3.2 | 传输加密 | 敏感数据传输是否使用 HTTPS |
| 3.3 | 数据脱敏 | 患者姓名、手机号、身份证等在日志/响应中是否脱敏 |
| 3.4 | 数据分类 | 是否明确哪些字段为敏感数据，并标注处理策略 |
| 3.5 | 数据删除 | 逻辑删除还是物理删除，删除后数据是否可恢复 |

## 4. 日志与审计

| # | 检查项 | 说明 |
|---|--------|------|
| 4.1 | 操作日志 | 关键业务操作（创建/修改/删除/导出）是否记录审计日志 |
| 4.2 | 日志脱敏 | 日志中是否避免打印密码、Token、敏感业务数据 |
| 4.3 | 日志级别 | 是否合理使用日志级别（不在生产环境打印 DEBUG） |
| 4.4 | 请求追踪 | 是否有 traceId 串联全链路日志 |

## 5. 接口安全

| # | 检查项 | 说明 |
|---|--------|------|
| 5.1 | 参数校验 | 所有入参是否做类型、长度、范围、格式校验 |
| 5.2 | 批量限制 | 批量接口是否设置单次数量上限（如 100 条） |
| 5.3 | 频率限制 | 敏感接口（登录、导出、发送通知）是否有频率限制 |
| 5.4 | 幂等设计 | 写接口是否考虑幂等性，防止重复提交 |
| 5.5 | 文件上传 | 上传接口是否校验文件类型、大小，是否有病毒扫描 |
| 5.6 | 导出安全 | 大量数据导出是否有权限控制和数量限制 |

## 6. 业务安全

| # | 检查项 | 说明 |
|---|--------|------|
| 6.1 | 并发控制 | 并发修改同一数据时是否有乐观锁/悲观锁保护 |
| 6.2 | 数据一致性 | 跨服务/跨表操作是否有事务或补偿机制 |
| 6.3 | 状态机安全 | 业务状态流转是否有合法性校验，防止非法跳转 |
| 6.4 | 金额/数量 | 涉及金额或数量计算时是否使用 DECIMAL，避免浮点精度问题 |
| 6.5 | 时间安全 | 是否使用服务端时间而非客户端传入时间做业务判断 |

## 7. 基础设施安全

| # | 检查项 | 说明 |
|---|--------|------|
| 7.1 | 配置安全 | 数据库密码、API Key 等是否使用配置中心/密钥管理，不硬编码 |
| 7.2 | 依赖安全 | 第三方依赖是否有已知漏洞，版本是否及时更新 |
| 7.3 | 错误处理 | 异常信息是否避免向前端暴露堆栈、SQL、内部路径 |
| 7.4 | CORS 配置 | 跨域配置是否限定允许的 Origin，避免使用 * |

```markdown
### 安全自查报告

**设计文档**：{文档标题}
**自查日期**：{YYYY-MM-DD}

| 类别 | 已覆盖 | 需补充 | 不适用 |
|------|--------|--------|--------|
| 注入防护 | {n} | {n} | {n} |
| 认证与授权 | {n} | {n} | {n} |
| 数据保护 | {n} | {n} | {n} |
| 日志与审计 | {n} | {n} | {n} |
| 接口安全 | {n} | {n} | {n} |
| 业务安全 | {n} | {n} | {n} |
| 基础设施安全 | {n} | {n} | {n} |

**需补充项详情**：

| # | 检查项 | 建议 |
|---|--------|------|
| {编号} | {检查项} | {具体建议} |
```
