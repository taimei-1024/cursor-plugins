# Design Language

This document defines the design language and visual principles that should be followed when creating wireframes and mockups.

## Visual Hierarchy

- **Primary Actions**: Use Ocean Blue (#0066CC) background with white text, 16px font size, 600 weight, 12px vertical padding, 24px horizontal padding, 8px border radius
- **Secondary Actions**: Use transparent background with Ocean Blue (#0066CC) border (2px solid) and Ocean Blue text, same padding and radius as primary
- **Information Architecture**: Use 24px spacing between major sections, 16px between related items, 8px between tightly grouped elements
- **Content Prioritization**: H1 headings use 32px font size, H2 use 24px, H3 use 20px; body text uses 16px

## Spacing & Layout

- **Grid System**:
  - Desktop: 12-column grid with 24px gutters, 1200px max container width
  - Tablet: 8-column grid with 16px gutters, 768px max container width
  - Mobile: 4-column grid with 16px gutters, full width with 16px side padding
- **Spacing Scale**:
  - 4px, 8px, 12px, 16px, 24px, 32px, 48px
- **Consistent Spacing**: Always use multiples of 4px

## Accessibility

- Normal text contrast ratio >= 4.5:1
- Large text contrast ratio >= 3:1
- Focus ring: 2px solid Ocean Blue (#0066CC), with visible offset
- Minimum touch target: 44x44px
